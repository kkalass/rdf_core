vocab/acl library

Web Access Control (ACL) Vocabulary

Provides constants for the Web Access Control (ACL) vocabulary, which is used for defining access control permissions in web resources.

Example usage:

import 'package:rdf_core/vocab/acl.dart';
final authorization = AclClasses.authorization;

All constants are pre-constructed as IriTerm objects to enable direct use in constructing RDF graphs without repeated string concatenation or term creation.
