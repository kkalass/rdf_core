DefaultTripleStorageStrategy

DefaultTripleStorageStrategy class final

Default implementation that organizes triples by subject IRI document

Triples are stored in the document specified by their subject IRI without the fragment. Subjects with blank nodes are stored in a special "orphaned" resource.
