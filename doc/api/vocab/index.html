vocab library

RDF Core Vocabulary Library

This library provides type-safe access to common RDF vocabularies and ontologies. It includes standardized terms from widely used vocabularies like FOAF, Dublin Core, Schema.org, and more.

By using these predefined terms, applications can build interoperable RDF data without needing to manually construct IRIs, reducing errors and improving code readability.
