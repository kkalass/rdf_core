TripleStorageStrategy

TripleStorageStrategy class abstract interface

Defines strategy for mapping RDF triples to storage locations in a Solid Pod

Implementations determine how triples are organized into files/resources within a Pod, enabling customized storage patterns while maintaining a consistent API.
