exceptions/rdf_parser_exception library

RDF Parser Exception Hierarchy

Defines a hierarchy of exceptions specific to RDF parsing operations, allowing applications to handle different types of parsing errors with fine-grained control.

Example usage:

import 'package:rdf_core/exceptions/rdf_parser_exception.dart';
try {
  // parse RDF data
} catch (e) {
  if (e is RdfParserException) print(e);
}

See also: RDF 1.1 Concepts - Syntax

Exceptions / Errors

RdfInvalidIriException
Exception thrown when the parser encounters an invalid IRI
RdfParserException
Base exception class for all RDF parser-related errors
RdfSyntaxException
Exception thrown when the parser encounters syntax errors in the input
RdfUnsupportedFeatureException
Exception thrown when the parser encounters an unsupported feature
