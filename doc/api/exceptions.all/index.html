exceptions/exceptions library

RDF Core Exceptions Library

Barrel file exporting all RDF exception types for easy import.

Example usage:

import 'package:rdf_core/exceptions/exceptions.dart';
try {
  // some RDF operation
} catch (e) {
  if (e is RdfException) print(e);
}

Classes

SourceLocation
Contains information about source location where an error occurred

Exceptions / Errors

RdfConstraintViolationException
Exception thrown when an RDF model constraint is violated
RdfCyclicGraphException
Exception thrown when the graph contains cycles that prevent serialization
RdfException
Base exception class for all RDF-related errors.
RdfInvalidIriException
Exception thrown when the parser encounters an invalid IRI
RdfParserException
Base exception class for all RDF parser-related errors
RdfSerializerException
Base exception class for all RDF serialization-related errors
RdfShapeValidationException
Exception thrown when an RDF graph doesn't conform to a specified shape
RdfSyntaxException
Exception thrown when the parser encounters syntax errors in the input
RdfTypeException
Exception thrown when there's a type error in RDF data
RdfUnsupportedFeatureException
Exception thrown when the parser encounters an unsupported feature
RdfUnsupportedSerializationFeatureException
Exception thrown when the serializer cannot represent a feature in the target format
RdfValidationException
Base exception class for all RDF validation-related errors
