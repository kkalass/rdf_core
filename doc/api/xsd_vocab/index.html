vocab/xsd library

XML Schema Definition (XSD) Vocabulary

Provides constants for the XML Schema Definition vocabulary, which defines data types used in RDF literals.

Example usage:

import 'package:rdf_core/vocab/xsd.dart';
final integerType = Xsd.integer;

All constants are pre-constructed as IriTerm objects to enable direct use in constructing RDF graphs without repeated string concatenation or term creation.
