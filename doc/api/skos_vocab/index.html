vocab/skos library

Simple Knowledge Organization System (SKOS) Vocabulary

Provides constants for the SKOS vocabulary, which is used for representing knowledge organization systems such as thesauri, classification schemes, subject heading lists and taxonomies.

Example usage:

import 'package:rdf_core/vocab/skos.dart';
final concept = SkosClasses.concept;

All constants are pre-constructed as IriTerm objects to enable direct use in constructing RDF graphs without repeated string concatenation or term creation.
