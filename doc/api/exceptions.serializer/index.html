exceptions/rdf_serializer_exception library

RDF Serializer Exception Hierarchy

Defines exceptions specific to RDF serialization operations, allowing applications to handle errors that occur when converting RDF graphs to specific serialization formats.

Example usage:

import 'package:rdf_core/exceptions/rdf_serializer_exception.dart';
try {
  // serialize RDF graph
} catch (e) {
  if (e is RdfSerializerException) print(e);
}

See also: RDF 1.1 Concepts - Serialization

Exceptions / Errors

RdfCyclicGraphException
Exception thrown when the graph contains cycles that prevent serialization
RdfSerializerException
Base exception class for all RDF serialization-related errors
RdfUnsupportedSerializationFeatureException
Exception thrown when the serializer cannot represent a feature in the target format
