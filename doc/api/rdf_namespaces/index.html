rdf_namespaces

vocab/namespaces library

RDF Namespace mappings

Defines standard mappings between RDF namespace prefixes and their corresponding URIs. These mappings are commonly used in RDF serialization formats like Turtle and JSON-LD.

Example usage:

import 'package:rdf_core/vocab/namespaces.dart';

// Get a namespace URI from a prefix
final rdfNamespace = rdfNamespaceMappings['rdf']; // http://www.w3.org/1999/02/22-rdf-syntax-ns#
