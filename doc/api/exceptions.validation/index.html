exceptions/rdf_validation_exception library

RDF Validation Exception Hierarchy

Defines exceptions for RDF data validation, allowing applications to handle cases where RDF data does not conform to expected constraints or data models. Focuses on semantic errors and structure issues.

Example usage:

import 'package:rdf_core/exceptions/rdf_validation_exception.dart';
try {
  // validate RDF graph
} catch (e) {
  if (e is RdfValidationException) print(e);
}

See also: SHACL - Shapes Constraint Language

Exceptions / Errors

RdfConstraintViolationException
Exception thrown when an RDF model constraint is violated
RdfShapeValidationException
Exception thrown when an RDF graph doesn't conform to a specified shape
RdfTypeException
Exception thrown when there's a type error in RDF data
RdfValidationException
Base exception class for all RDF validation-related errors
