RdfConstants

RdfConstants class

Centralized repository of common RDF constants and vocabulary terms.

This class contains pre-initialized IRI term constants for core RDF vocabulary elements defined in the RDF 1.1 specification. These constants are essential for creating well-formed RDF graphs that adhere to the standard.

The constants are implemented as static members on a non-instantiable class to provide a convenient namespace without requiring instantiation.
