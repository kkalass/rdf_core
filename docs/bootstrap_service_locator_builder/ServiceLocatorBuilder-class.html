ServiceLocatorBuilder

ServiceLocatorBuilder class

Provides a fluent API for building and configuring the service locator. The idea here is, that we create extensions for this builder and allow extensions to add their own services to the locator.

Available extensions
  • AuthServiceLocatorBuilderExtension
  • CoreServiceLocatorBuilderExtension
  • RdfMappingServiceLocatorBuilderExtension
  • RepositoryServiceLocatorBuilderExtension
  • StorageServiceLocatorBuilderExtension
  • SyncableRepositoryServiceLocatorBuilderExtension
  • SyncServiceLocatorBuilderExtension
